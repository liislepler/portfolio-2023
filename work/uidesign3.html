Trackersaurus

Trackersaurus is an application for tracking activities, which features different user levels for parents and children. The project was done as part of the course Applied Web Architecture. The frontend of the application was done with using PHP connected with backend using REST API and a MySQL database. The project consists of 3 layers: api, business-logic, and presentation layer.

Co-creator: Elin Tri
