iCountime

Work Management App

iCountime is a work management app with features to help freelance-working people plan and keep track of projects, work hours, and expenses in a time-efficient way. The project was made as part of the course Interaction Design. The hi-fi wireframes with animations were done on Figma and user-tested.

Co-creators: Elin Tri, Alicia Liu

Note App

Note app provides a platform for people to share knowledge connected to different courses, through note-taking. The project was done as part of the course Client-Server Communication. The frontend of the application was done with Vue.js JavaScript framework connected with backend using REST API and a SQLite database. The backend was structure with Express.js using Node.js.

Co-creator: Elin Tri
